# 「1時間あたり」と「縫う以外の時間」の説明書き

現場のみなさんへ。アプリに新しく入った2つのことについての説明です。
読むのに3分かかりません。

---

## 1. 大きな青い数字は「1時間あたり」です

作業記録の画面で自分の名前を選ぶと、大きな青い数字が出ます。

これは **「今日、1時間でいくらぶんの仕事をしたか」** です。

### 計算のしかた

```
今日つくった価値 ÷ 縫うのに使った時間 ＝ 1時間あたり

例:  ¥42,000 ÷ 7時間 ＝ ¥6,000
```

### なぜ「合計」ではなく「1時間あたり」なの？

長い時間働いた人の数字が大きくなるのではなく、
**時間を上手に使った人の数字が大きくなる**ようにするためです。

- 10時間かけて ¥50,000 → 1時間あたり ¥5,000
- 7時間で ¥42,000 → 1時間あたり **¥6,000** ← こちらのほうが上

急いでたくさんやることより、**無理なく・無駄なく・段取りよく**。
それがこの数字の伸ばし方です。

### どこで見られる？

- **作業記録の画面**: 名前を選ぶと大きな数字で表示。記録するたびに更新されます
- **直近1週間のグラフ**: 数字の下の棒グラフ。「1時間あたり／金額／枚数」を切り替えて、日々の変化が見られます
- 記録すると、その日にあった一言が出ます。8時間以上がんばった日は「無理は禁物です」と出ます。ちゃんと休んでください

## 2. 縫う以外の仕事をした日は、チェックを入れてください

芯を貼った。裁断をした。だれかの作業を手伝った。準備や段取りをした。

こういう**「縫う以外の仕事」も、服をつくるために欠かせない大事な仕事**です。
ただし、この時間まで「縫った時間」として計算すると、あなたの1時間あたりが
**本当より悪く見えてしまいます**。

そこで、縫う以外の仕事をした日は、その時間を分けて教えてください。

### 入れ方（3ステップ）

1. いつもどおり、**品番**と**作業時間**を入れる（例: 8時間）
2. 作業時間の下にある
   **「☐ 工程以外の作業があった（芯貼り・裁断・サポートなど）」**
   にチェックを入れる
3. 「8時間のうち [　] 時間」と出るので、縫う以外に使った時間を入れる（例: 1）

あとはいつもどおり記録ボタンを押すだけです。

### 入れるとどうなる？

```
作業時間 8時間（うち 縫う以外 1時間）で ¥42,000 の場合

× チェックなし: ¥42,000 ÷ 8時間 ＝ ¥5,250（損して見える）
○ チェックあり: ¥42,000 ÷ 7時間 ＝ ¥6,000（正しい数字）
```

- あなたの1時間あたりは、**縫った7時間だけ**で正しく計算されます
- 働いた8時間は、そのまま記録に残ります（消えません）
- 手伝いや準備の時間も「見える仕事」として記録されます。
  **隠す必要も、遠慮する必要もありません。**

### よくある質問

**Q. 何分くらいから入れればいい？**
だいたい合計30分〜1時間以上あった日は入れてください。数分の手伝いまで細かく数えなくて大丈夫です。

**Q. 手伝いのとき、品番はどれを選ぶ？**
自分がそのとき作業していた品番のままでOKです。

**Q. チェックを入れ忘れた！**
次の日から入れてくれれば大丈夫です。気になる場合はリーダーに相談してください。
